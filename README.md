## Tags
### Основные теги
```
<html> 
    <head>
    Описание сайта/метаданные/другие данные 
     
    </head>
    <body>
    Описание тела страницы
    </body>
</html>
```
#### Тэги наследования head
`<title>заглавие</title>`

#### Тэги наследования body
`<p>параграф paragraph</p>`
